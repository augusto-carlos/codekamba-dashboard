## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

### Frontend
- [ReactJS](https://reactjs.org)
- [AxiosJS](https://github.com/axios/axios)
- [AlertifyJS](https://alertifyjs.com/)

### Backend
- [NodeJS/AdonisJS](https://adonisjs.com/)
- JWT (JasonWebToken)


## 💻 Projeto

O Codetrial dashboard é um projeto desenvolvido para acompanhar o fluxo de informações dentro de um blog pessoal.Podendo assim:

 - Gestão de conteúdos
 - Monitorização dos serviços de terceiros
 - Gestão direita do banco de dados

## 🤔 Como contribuir

- Faça um fork desse repositório;
- Cria uma branch com a sua feature: `git checkout -b minha-feature`;
- Faça commit das suas alterações: `git commit -m 'feat: Minha nova feature'`;
- Faça push para a sua branch: `git push origin minha-feature`.

Depois que o merge da sua pull request for feito, você pode deletar a sua branch.
